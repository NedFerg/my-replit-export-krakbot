2. Add helper to compute execution price

Still in simulation.py, add this method:

def apply_microstructure(self, mid_price, side):
    """
    side: 'buy' or 'sell'
    Returns execution_price, total_cost
    """
    spread = mid_price * (self.spread_bps / 10000)
    slippage = mid_price * (self.slippage_bps / 10000)
    txn_cost = mid_price * (self.txn_cost_bps / 10000)

    if side == "buy":
        exec_price = mid_price + spread/2 + slippage
    else:
        exec_price = mid_price - spread/2 - slippage

    return exec_price, txn_cost

This gives a realistic execution price and a per‑trade cost.
