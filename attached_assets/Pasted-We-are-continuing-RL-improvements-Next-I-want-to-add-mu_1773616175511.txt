1. Add rolling price history in the simulator

In simulation.py, inside the simulation class __init__, add:
python

self.trend_windows = [5, 20, 50]
self.price_history = []

(If price_history already exists, reuse it — just ensure it stores raw prices each step.)
2. Compute multi‑timeframe momentum each step

Inside the main step loop (the same place where volatility and drawdown are computed), after updating the current price, add:
python

# Track raw prices
self.price_history.append(current_price)

# Compute multi-timeframe momentum
momentums = {}
for w in self.trend_windows:
    if len(self.price_history) > w:
        past_price = self.price_history[-w-1]
        if past_price != 0:
            mom = (current_price - past_price) / past_price
        else:
            mom = 0.0
    else:
        mom = 0.0
    momentums[w] = mom

# Attach to state
state.mom_5 = momentums[5]
state.mom_20 = momentums[20]
state.mom_50 = momentums[50]

These are simple percentage‑change momentums over different horizons.
