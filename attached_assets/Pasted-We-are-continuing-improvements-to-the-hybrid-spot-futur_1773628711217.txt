Goal:  
Add real funding‑rate integration so the futures overlay automatically reduces exposure when funding is expensive and increases exposure when funding is favorable.

This protects the system during euphoric altseason (when funding can become extremely costly) and enhances returns during negative‑funding environments.
1. Add funding‑rate smoothing state

In broker.__init__, add:

# Funding rate smoothing (EMA)
self.funding_ema = {}
self.funding_alpha = 0.2  # smoothing factor

2. Add funding‑rate update helper

Place this anywhere in broker.py:
python

def update_funding_rate(self, asset, raw_funding):
    # Initialize if needed
    if asset not in self.funding_ema:
        self.funding_ema[asset] = raw_funding
    else:
        # EMA smoothing
        prev = self.funding_ema[asset]
        self.funding_ema[asset] = prev + self.funding_alpha * (raw_funding - prev)

This allows you to feed real funding rates later (from exchange API or simulation).

# --- Funding-aware adjustment ---
